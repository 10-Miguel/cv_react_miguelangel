Pasos para ejecutar el proyecto
1.Clonar el repositorio:
git clone https://github.com/10-Miguel/cv_react_miguelangel

Entrar al directorio del proyecto:
cd cv_react_miguelangel

Instalar dependencias:
npm install

Ejecutar el servidor de desarrollo:
npm run dev
Abrir en el navegador el enlace que aparece:
http://localhost:5173
